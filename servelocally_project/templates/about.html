ServeLocally is an Open Source project on Github. It was originally created for WeServe South Eastern Ohio by Kelly Creative Tech.

It's built on Django, and can be used in any Django Project. We have plans to have an entire example project completed so you can get up and running very quickly

The technology is really cool, but it takes a much more concerted effort to get a volunteer match site full, and in use in your community. We're going through the process now in South Eastern Ohio, along with WeServe SEO, Local Sponsors, and several community groups. We'd be happy to help you in your community

Send us an Email at info@servelocally.org if you have any questions or comments
