ServeLocally is an Open Source project on Github. It was originally created
for weServe Southeastern Ohio by Kelly Creative Tech.

ServeLocally is built on Django, and can be used in any Django Project. We have plans to have an entire example project completed so you can get up and running as quickly as possible.

The technology is really great, but it takes a much more concerted effort to get a volunteer match site fully populated, and in use in your community.

weServe Southeastern Ohio is going through the process now to generate interest among community organizations and volunteer groups, as well as soliciting financial support for the ongoing maintenance and administrative costs for the software.

We are documenting the process to make it easier to get started, and we'd be happy to help you in your community.

Send us an Email at info@servelocally.org if you have any questions or comments.
