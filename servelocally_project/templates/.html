About ServeLocally

ServeLocally is an open source project of Kelly Creative Tech. It was originally created for weServe South Eastern Ohio. You can build an instance for your community, or we can help you get setup, and on the path to having a well maintained community site.

Features

  • Easy to Update: This is the fist goal, it was meant to be as easy to use as email.
  • Easy to Use: You don't need a special account to browse opportunities. We have SMS and Telephone services so the database is accessible to volunteers with limited internet access or skill
  • Many more

Easy to get started and Grow

ServeLocally is built on Django and Python Many developers have those skills, and we're building documentation on how to get started.
